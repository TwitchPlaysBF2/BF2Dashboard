Q: What is joinme.click?
A: It's the software that makes it possible to join BF2 from the browser, by using the [Join] button on BF2.TV

Q: How to configure the joinme.click launcher to open the game in a windowed state?
A: Change the filename of `config.windowed-example.yaml` to `config.yaml` for the config to be effective and the game will launch windowed.
